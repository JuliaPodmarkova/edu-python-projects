## рҹҡҖ РҡР°Рә РҝРҫСҒРјРҫСӮСҖРөСӮСҢ Рё Р·Р°РҝСғСҒСӮРёСӮСҢ

1. ### РҡР»РҫРҪРёСҖРҫРІР°СӮСҢ:
   ```bash
   git clone https://github.com/JuliaPodmarkova/edu-python-projects.git
   cd edu-python-projects ```
   
2. ### РҹРөСҖРөР№СӮРё РІ РҝР°РҝРәСғ РәРҫРҪРәСҖРөСӮРҪРҫРіРҫ РҝСҖРҫРөРәСӮР°, РҪР°РҝСҖРёРјРөСҖ ModulsOfTema1/Module1

3. ### Р—Р°РҝСғСҒСӮРёСӮСҢ СҒРәСҖРёРҝСӮ:
   ```bash
   python script_name.py
    ```
   
   (РҳРјСҸ СҒРәСҖРёРҝСӮР° СҒРјРҫСӮСҖРёСӮРө РІРҪСғСӮСҖРё РҝР°РҝРәРё вҖ” РҫРұСӢСҮРҪРҫ main.py РёР»Рё task.py.)

## рҹ”Қ Р§СӮРҫ СӮСғСӮ РјРҫР¶РҪРҫ СғРІРёРҙРөСӮСҢ

**Р РөСҲРөРҪРёРө** СӮРёРҝРҫРІСӢС… Р°Р»РіРҫСҖРёСӮРјРёСҮРөСҒРәРёС… Р·Р°РҙР°СҮ.

**Р Р°РұРҫСӮСғ** СҒ С„Р°Р№Р»Р°РјРё, СҒСӮСҖРҫРәР°РјРё, СҒРҝРёСҒРәР°РјРё.

**РЎРҫРұР»СҺРҙРөРҪРёРө** PEP8 Рё РҙРҫРәСғРјРөРҪСӮРёСҖРҫРІР°РҪРёРө РәРҫРҙР°.

**Р РҫСҒСӮ** СғСҖРҫРІРҪСҸ РҫСӮ РҝСҖРҫРөРәСӮР° Рә РҝСҖРҫРөРәСӮСғ.
