# edu-python-projects

[![Python](https://img.shields.io/badge/python-3.10%2B-blue)]()

Учебные проекты (домашние задания) по Python, выполненные в рамках курса Urban University.  
Репозиторий демонстрирует мои навыки написания чистого кода, оформления документации и владения Git.

---

## 📁 Структура

edu-python-projects/
├── HomeWork1/
│ └── ... (решение задачи №1)
├── HomeWork2/
│ └── ... (решение задачи №2)
├── VISION.md # цель создания и ценность проектов
├── REQUIREMENTS.md # технические требования к коду и оформлению
├── USER_STORIES.md # для кого и зачем всё это
├── ROADMAP.md # планы по улучшению
└── README.md


## 🚀 Как посмотреть и запустить

1. # Клонировать:
   ```bash
   git clone https://github.com/JuliaPodmarkova/edu-python-projects.git
   cd edu-python-projects ```
   
2. # Перейти в папку конкретного проекта, например HomeWork1.

3. # Запустить скрипт:
   ```bash
   python script_name.py ```
   
   (Имя скрипта смотрите внутри папки — обычно main.py или task.py.)

## 🔍 Что тут можно увидеть

''Решение'' типовых алгоритмических задач.
