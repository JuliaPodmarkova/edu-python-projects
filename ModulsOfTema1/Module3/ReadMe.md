# Модуль 3: функции, рекурсия, работа со структурами данных, графический интерфейс (Tkinter)

В этой папке собраны программы, которые углубляют понимание функций (пространства имён, параметры по умолчанию, произвольное число аргументов, распаковка), рекурсии, а также демонстрируют простой графический интерфейс на Tkinter.

## 📂 Содержание

| Файл | Что делает | Демонстрирует |
|------|------------|----------------|
| `Calculator.py` | Простой графический калькулятор с кнопками +, -, *, / | Tkinter, события кнопок, получение данных из полей ввода, размещение виджетов |
| `Lectures.py` | Учебные примеры: локальные/глобальные переменные, параметры по умолчанию, *args, **kwargs, рекурсия, встроенные функции (`any`, `all`, `zip`, `isinstance` и др.) | пространства имён, изменяемые значения по умолчанию (list), рекурсия (сумма чисел), практика нахождения максимума, чётных, уникальных элементов |
| `module3_hurd.py` | Рекурсивный обход вложенной структуры данных (списки, словари, кортежи, строки, числа) с подсчётом суммы: для чисел прибавляется само число, для строк — их длина | рекурсия, обработка разных типов (`isinstance`), работа с `dict.items()`, обход произвольной вложенности |
| `module_3_4.py` | Поиск слов, содержащих корневое слово (регистронезависимо). Например, 'rich' → 'richiest', 'orichalcum', 'richies' | строки, приведение к нижнему регистру, проверка вхождения (`in`), упаковка аргументов (`*other_words`) |

## 🚀 Как запустить любой скрипт

1. Откройте терминал в папке `ModulsOfTema1/Module2`.
2. Выполните:
   ```bash
   python имя_файла.py
   ```

## 📦 Зависимости
Calculator.py требует установленную библиотеку tkinter (входит в стандартную поставку Python, но на некоторых Linux может потребоваться python3-tk).

Остальные скрипты используют только стандартную библиотеку.
