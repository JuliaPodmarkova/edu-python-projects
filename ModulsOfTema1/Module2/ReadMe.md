# Модуль 2: структуры данных, функции, условия, игра «Крестики-нолики»

В этой папке собраны учебные программы, демонстрирующие работу с кортежами, списками, строками, функциями, условиями, циклами и небольшой игровой проект.

## 📂 Содержание

| Файл | Что делает | Демонстрирует |
|------|------------|----------------|
| `Cortedj.py` | Создание кортежей разными способами, сравнение размера списка и кортежа, изменение вложенного списка, конкатенация и умножение кортежей | кортежи (`tuple`), неизменяемость, методы `__sizeof__`, индексация |
| `Str.py` | Запрашивает имя и год рождения, вычисляет возраст, даёт разные ответы в зависимости от возраста | ввод (`input`), преобразование типов, условные операторы `if-elif-else`, f-строки |
| `TicTacToe.py` | Игра «Крестики-нолики» для двух игроков (по очереди вводят координаты) | двумерные списки, функции, проверка победителя, вложенные условия, игровой цикл |
| `def.py` | Примеры функций: лотерея (случайный выбор), двухдневная лотерея (без повторений), демонстрация параметров по умолчанию и распаковки | функции (`def`), `random.choice()`, `list.remove()`, параметры по умолчанию, распаковка `*` |
| `lists.py` | Работа со списками: замена элементов, добавление (`.append()`, `.extend()`), срезы, удаление элементов, проверка наличия (`in`, `not in`) | списки (`list`), методы списков, срезы |
| `module_2_3.py` | Проходит по списку чисел, выводит положительные, пропускает нули, останавливается на отрицательном | цикл `while`, `break`, `continue`, индексация |
| `module_2_hard.py` | Генерация пар чисел, сумма которых является делителем заданного числа (от 3 до 20). Игровая обёртка с поиском подсказок | вложенные циклы, проверка делимости (`%`), генерация строк, пауза (`time.sleep()`) |

## 🚀 Как запустить любой скрипт

1. Откройте терминал в папке `ModulsOfTema1/Module2` (где лежит этот README).
2. Выполните:
   ```bash
   python имя_файла.py
   ```
   
## 📦 Зависимости
Все скрипты используют только стандартную библиотеку Python. Дополнительных установок не требуется.
