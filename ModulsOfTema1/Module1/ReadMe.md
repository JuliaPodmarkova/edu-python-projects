# Модуль 1: основы Python

В этой папке собраны первые учебные программы, написанные в рамках курса.  
Они охватывают базовые конструкции: арифметические операции, сравнения, работу со строками, списками, словарями, ввод/вывод.

## 📂 Содержание

| Файл | Что делает | Демонстрирует |
|------|------------|----------------|
| `1st program.py` | Возводит 9 в степень 0.5, затем умножает на 5 | арифметика, корень (`**`), точка с запятой (не PEP8) |
| `2nd program.py` | Сравнивает числа: `9.99 > 9.98 and 1000 != 1000.1` | логические операторы, вывод `bool` |
| `3rd program.py` | Сравнивает `2*2+2` и `2*(2+2)`, выводит оба результата и их равенство | приоритет операций, вывод чисел и `bool` |
| `4th program.py` | Извлекает первую дробную цифру из числа `123.456` | преобразование строки в `float`, арифметика с плавающей точкой |
| `Modul1PracticTask.py` | Считает средний балл студентов по списку оценок и выводит словарь | списки, множества, сортировка, `zip()`, `dict`, округление |
| `module_1_4.py` | Работа с пользовательским вводом: длина строки, регистры, удаление пробелов, первый и последний символ | строки, методы `.upper()/lower()/replace()`, индексация |

## 🚀 Как запустить любой скрипт

1. Откройте терминал в папке `ModulsOfTema1/`.
2. Выполните:
   ```bash
   python Module1/имя_файла.py
   ```
## 📦 Зависимости
Нет внешних библиотек. Нужен только Python 3.10+.
