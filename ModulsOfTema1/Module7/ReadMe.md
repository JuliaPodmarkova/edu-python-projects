# Модуль 7: файлы, модуль os, форматирование строк, работа с текстами

В этой папке собраны программы, демонстрирующие взаимодействие с операционной системой (`os`), чтение и запись файлов, позиционирование внутри файлов, менеджер контекста `with`, различные способы форматирования строк (конкатенация, `%`, `.format()`, f-строки), а также работу с байтами и строками (кодировки). В качестве прикладных задач реализованы простой проводник на Tkinter и класс `WordsFinder` для поиска слов в текстовых файлах.

## 📂 Содержание

| Файл / Папка | Что делает | Демонстрирует |
|--------------|------------|----------------|
| `os_module.py` | Работа с модулем `os`: текущая директория, создание папок, `os.walk`, фильтрация файлов/папок, статистика о файлах, выполнение системных команд. | `os.getcwd()`, `os.mkdir`, `os.chdir`, `os.path.exists`, `os.listdir`, `os.walk`, `os.path.isfile`, `os.stat`, `os.system` |
| `HomeWork7.py` | Рекурсивный обход всех файлов в текущей директории (и поддиректориях) с помощью `os.walk`. Выводит имя файла, полный путь, размер, время изменения, родительскую директорию. | `os.walk`, `os.path.getmtime()`, `os.path.getsize()`, `os.path.dirname`, `time.strftime` |
| `explorer.py` | Простой графический проводник на Tkinter: открывает диалоговое окно выбора файла и запускает его (`os.startfile`). | `tkinter`, `filedialog`, `os.startfile`, GUI |
| `readFilesMode.py` | Чтение (`r`), запись (`w`), дозапись (`a`) в текстовые файлы. Использование `seek()` и `tell()` для перемещения курсора. | режимы открытия файлов, `read()`, `write()`, `seek()`, `tell()` |
| `positioningInTheFile.py` | Работа с `seek()` и `tell()` на примере добавления текста в середину файла (после отступа). | позиционирование в файле, чтение с определённой позиции |
| `theWithOperator.py` | Использование менеджера контекста `with` для автоматического закрытия файла. Чтение файла посимвольно. | `with open(...) as file`, построчное и посимвольное чтение |
| `StrFormat.py` | Все способы форматирования строк: конкатенация, `%` (старый стиль), `.format()`, f-строки. | форматирование строк, экранирование, именованные параметры |
| `strAndByts.py` | Перевод символов в их коды ASCII/Unicode (`ord`, `chr`), получение байтового представления (`b'\x68'`), декодирование. | `ord`, `chr`, байтовые литералы, `decode()` |
| `Module7HW.py` | Класс `WordsFinder` для анализа нескольких текстовых файлов: приведение к нижнему регистру, удаление пунктуации, разбивка на слова, поиск первого вхождения слова и подсчёт количества вхождений. | работа со списком файлов, методы строк (`replace`, `split`, `lower`), циклы, словари |
| Тестовые текстовые файлы | Примеры текстов для `WordsFinder`: `test_file.txt`, `Mother_Goose_Mondays_Child.txt`, `RudyardKipling_If.txt`, `Walt_Whitman _O_Captain!_My_Captain!.txt`, `sample.txt`, `sample2.txt`, `sample3.txt` | вспомогательные данные |
| `Module7_mode_with/` | Папка с результатами работы `WordsFinder` (выводы и примеры использования). | демонстрация работы класса |

## 🚀 Как запустить любой скрипт

1. Откройте терминал в папке `ModulsOfTema1/Module7`.
2. Выполните:
   ```bash
   python os_module.py
   python HomeWork7.py
   python explorer.py         # (откроется окно Tkinter)
   python readFilesMode.py
   python positioningInTheFile.py
   python theWithOperator.py
   python StrFormat.py
   python strAndByts.py
   python Module7HW.py
   ```
Примечание: explorer.py требует установленной библиотеки tkinter (входит в стандартную поставку Python, но на Linux может потребоваться python3-tk).
В os_module.py некоторые строки закомментированы (создание папок, запуск файлов) – снимите комментарии при необходимости.

## 📦 Зависимости
Все скрипты используют только стандартную библиотеку Python.

Для работы os_module.py и HomeWork7.py достаточно прав на чтение файловой системы.

explorer.py требует tkinter (обычно уже есть).
