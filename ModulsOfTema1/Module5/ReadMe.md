# Модуль 5: объектно-ориентированное программирование (ООП)

В этой папке собраны программы, демонстрирующие ключевые концепции ООП в Python: создание классов и объектов, конструкторы, магические (dunder) методы, области видимости атрибутов, паттерн «одиночка» (singleton), а также небольшие прикладные проекты (система регистрации, упрощённый YouTube-подобный сервис).

## 📂 Содержание

| Файл | Что делает | Демонстрирует |
|------|------------|----------------|
| `Classes.py` | Класс `Human` с атрибутами имени, фамилии, возраста. Магические методы: `__del__` (вызывается при удалении), `__len__` (возвращает возраст), `__lt__/__gt__/__eq__` (сравнение по возрасту и имени), `__str__`. В конструкторе вызываются `say_info()`, `birthday()`. | базовое ООП, магические методы, изменение атрибутов экземпляра и класса (`head`) |
| `color.py` | Две функции для цветного вывода в терминал: `out_red(text)`, `out_green(text)`. Используют ANSI-коды. | вспомогательный модуль, форматирование вывода |
| `method_new.py` | Демонстрация паттерна «одиночка» (singleton) через переопределение `__new__`. Класс `User` всегда возвращает один и тот же экземпляр. Также показана распаковка аргументов (`*args`, `**kwargs`) в `__init__`. | `__new__`, управление созданием экземпляров, `super()`, `__instance` |
| `module_5_3.py` | Класс `House` с атрибутами названия и этажности. Реализованы все магические методы сравнения (`__eq__`, `__lt__`, `__le__`, `__gt__`, `__ge__`, `__ne__`) и арифметические (`__add__`, `__radd__`, `__iadd__`) для увеличения этажности. | перегрузка операторов, `isinstance`, `NotImplemented` |
| `practic_In_Module.py` | Система регистрации и входа пользователей. Классы `User` и `Database`. Проверка пароля на длину, наличие цифр, заглавных и строчных букв. Используется `color.py` для цветного вывода сообщений. | ООП в прикладной задаче, инкапсуляция, валидация, работа со словарём как БД |
| `module5hard.py` | Упрощённая модель видеохостинга (аналог YouTube). Классы: `User` (ник, хеш пароля, возраст), `Video` (название, длительность, adult_mode), `UrTube` (список пользователей, видео, текущий пользователь). Методы: регистрация, вход, добавление видео, поиск, воспроизведение с учётом возрастных ограничений. | сложный пример ООП, хеширование (`hashlib`), работа с состоянием, имитация потокового воспроизведения (`time.sleep`) |

## 🚀 Как запустить любой скрипт

1. Откройте терминал в папке `ModulsOfTema1/Module5`.
2. Выполните:
   ```bash
   python Classes.py
   python method_new.py
   python module_5_3.py
   python practic_In_Module.py
   python module5hard.py
   ```

> **Примечание**: Для `practic_In_Module.py` требуется файл `color.py` в той же папке (или в пакете `Module5`, как указано в импорте). Убедитесь, что структура соответствует.

## 📦 Зависимости

- Все скрипты используют только стандартную библиотеку Python.
- `module5hard.py` использует модули `hashlib` и `time`.
- `practic_In_Module.py` использует `random.choice` (импортирован, но не задействован явно) и `color.py`.
