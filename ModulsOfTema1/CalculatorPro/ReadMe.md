# Calculator Pro

Простой, но функциональный графический калькулятор на Python с использованием Tkinter.  
Поддерживает базовые арифметические операции, тригонометрические функции, возведение в степень, факториал, квадратный корень, число π и смену знака.

## Возможности

- Сложение, вычитание, умножение, деление
- Возведение в степень (`xⁿ` → `**`)
- Квадратный корень (`√`)
- Синус и косинус (`sin`, `cos`)
- Факториал (формат: `5n!`)
- Число π (`π`)
- Смена знака числа (`±`)
- Скобки для группировки выражений
- Кнопка `C` для очистки поля ввода
- Кнопка `Exit` для выхода

## Установка и запуск

### Требования

- Python 3.6 или выше
- Модуль `tkinter` (входит в стандартную библиотеку Python, но требует наличия системной библиотеки Tk)

### Установка Tkinter (если необходимо)

#### Linux (Arch Linux)
```bash
sudo pacman -S tk
```

#### Linux (Ubuntu/Debian)
```bash
sudo apt-get install python3-tk
```

#### Windows
Tkinter обычно уже установлен вместе с Python. Если нет – переустановите Python, убедившись, что флажок `tcl/tk` отмечен.

#### macOS
```bash
brew install python-tk
```

### Запуск

1. Скачайте файл `CalculatorPro.py` в любую папку.
2. Откройте терминал (командную строку) в этой папке.
3. Выполните команду:
   ```bash
   python3 CalculatorPro.py
   ```

## Использование

1. Вводите выражение, нажимая кнопки калькулятора.
2. Для вычисления нажмите `=`.
3. Результат появится в поле ввода.
4. Для нового выражения нажмите `C` и введите следующий пример.

### Примеры выражений

| Ввод | Результат |
|------|-----------|
| `2+2*2` | 6 |
| `(2+2)*2` | 8 |
| `3xⁿ2` | 9 |
| `sqrt(16)` | 4.0 |
| `sin(π/2)` | 1.0 |
| `5n!` | 120 |
| `-5+3` | -2 |
| `10/3` | 3.3333333333333335 |

## Примечания

- Для факториала запись должна быть строго в формате `[число]n!`, например `5n!`. Поддерживаются только целые числа.
- Тригонометрические функции `sin` и `cos` ожидают аргумент в радианах.
- Для возведения в степень используйте кнопку `xⁿ` – она вставляет `**`.
- Кнопка `±` меняет знак числа, стоящего в начале строки. Для изменения знака результата или числа в середине выражения используйте скобки или введите `-` вручную.

## Ошибки

- Если выражение составлено неверно (например, нарушен синтаксис, деление на ноль), появится окно с сообщением об ошибке. Поле ввода будет очищено.
- Не используйте буквы или недопустимые символы – только цифры, операторы, скобки и функции, предоставленные интерфейсом.
