# Модуль 6: наследование, инкапсуляция, полиморфизм, MRO, игровой проект

В этой папке собраны программы, углубляющие ООП: наследование (одинарное и множественное), инкапсуляцию (приватные атрибуты), магические методы, порядок разрешения методов (MRO), а также практический проект — игра «Pong» на библиотеке Arcade.

## 📂 Содержание

| Файл / Папка | Что делает | Демонстрирует |
|--------------|------------|----------------|
| `Game/game.py` | Полноценная игра «Pong» (аналог настольного тенниса) с использованием библиотеки `arcade`. Управление платформой стрелками влево/вправо, мяч отскакивает от стен и платформы. | создание игрового цикла, работа со спрайтами, обработка ввода, коллизии, наследование от `arcade.Window`, переопределение `update`, `on_draw`, `on_key_press/release` |
| `inheritance.py` | Демонстрация уровней доступа: публичные атрибуты (`head`), защищённые (`_legs`), приватные (`__arms`). Классы `Student` и `Teacher` наследуют от `Human`. Показан доступ к приватному атрибуту через `_Human__arms` (name mangling). | инкапсуляция, name mangling, переопределение методов, `dir()`, наследование |
| `multiple_inheritance.py` | Классы `Human`, `StudentGroup`, `Student` (наследует от обоих). В конструкторе `Student` вызывается `super().__init__` для передачи имени и группы. Показан порядок MRO. | множественное наследование, `super()`, MRO, передача параметров в родительские классы |
| `module6hard.py` | Иерархия геометрических фигур: `Figure` (базовый), `Circle`, `Triangle`, `Cube`. Реализованы проверки цвета (RGB), количества сторон, вычисление площади, объёма, периметра. Используются приватные атрибуты (`__color`, `__sides`), магические методы (`__len__`). | абстрактные концепции, валидация входных данных, наследование, полиморфизм, инкапсуляция, математические расчёты |
| `module_6_3.py` | Иерархия животных: `Animal` -> `Birds`, `AquaticAnimal`, `PoisonousAnimal` -> `Duckbill` (утконос, наследующий от трёх классов). Показаны методы `move` (с учётом глубины), `dive_in`, `lay_eggs`, `attack` (в зависимости от `_DEGREE_OF_DANGER`). | множественное наследование, переопределение атрибутов, работа с координатами, `random` |

## 🚀 Как запустить любой скрипт

1. Откройте терминал в папке `ModulsOfTema6/`.
2. Для запуска:
   ```bash
   python Game/game.py          # требует установки arcade
   python inheritance.py
   python multiple_inheritance.py
   python module6hard.py
   python module_6_3.py
   ```
Важно: Game/game.py требует установки библиотеки arcade. Установите её командой:

    ```bash
    pip install arcade
    ```

## 📦 Зависимости
Game/game.py — требует arcade (и возможно pyglet, но arcade установит сам).

Остальные скрипты используют только стандартную библиотеку (random, math, tokenize не используется в multiple_inheritance.py — импорт лишний).
