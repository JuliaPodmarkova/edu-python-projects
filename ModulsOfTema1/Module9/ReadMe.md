# Модуль 9: функциональное программирование, итераторы, генераторы, декораторы

В этой папке собраны программы, демонстрирующие функциональные возможности Python: передача функций как объектов, `map`, `filter`, генераторы списков/словарей/множеств, итераторы (классы с `__iter__` и `__next__`), функции-генераторы (`yield`), бесконечные генераторы, замыкания, декораторы (с параметрами и без), мемоизация, обработка больших файлов построчно.

## 📂 Содержание

| Файл | Что делает | Демонстрирует |
|------|------------|----------------|
| `Lection1.py` | Передача функций в качестве аргументов (`adder`, `multiplier`), использование `map` и `filter` для списка чисел. | функции высшего порядка, `map`, `filter`, `list(map(...))` |
| `Lection2.py` | Генераторы списков, множеств, словарей. Условные выражения внутри генераторов. | list comprehension, set comprehension, dict comprehension, тернарные операторы |
| `Lection3.py` | Генераторные выражения (`(x**100 for x in my_numbers)`), их одноразовость, сравнение скорости с list comprehension. Функции `range`, `zip`, `map` как итераторы. | генераторные выражения, ленивые вычисления, `time`, `zip`, `map` |
| `Lection4.py` | Лямбда-функции, замыкания (функция `get_multiplier_v1` возвращает функцию), класс с методом `__call__` (функтор). | `lambda`, замыкания, `__call__`, динамическое создание функций |
| `Lection5.py` | Итераторы из модуля `itertools` (`repeat`, `product`), сравнение размера итератора и строки. Свой класс итератора `Iter`. Класс `Fibonacci` (с ошибками). | `itertools`, память итераторов, `__iter__`, `__next__`, `StopIteration` |
| `Lection6.py` | Функции-генераторы (`yield`), сравнение с возвратом списка (фибоначчи). Бесконечный генератор. Чтение большого файла построчно (`yield`). | `yield`, генераторы, экономия памяти, обработка больших файлов |
| `Lection7_decorators.py` | Простейший декоратор `null_decorator` (вызывает функцию и замеряет время), декоратор `uppercase`, декоратор `time_track` для замера времени выполнения. | декораторы (синтаксис `@`), время выполнения, обёртки |
| `Lection8_decorators.py` | Декоратор с параметром (`func_gen_dec`), создающий декоратор с заданной точностью замера времени. | декораторы с параметрами, фабрики декораторов |
| `Module9_HW_1.py` | Класс `Iterator`, повторяющий поведение `range()` с проверкой шага (шаг не может быть 0 – выбрасывается `StepValueError`). | пользовательские исключения, итераторы, проверка границ |
| `Module9_HW_2.py` | Декоратор `is_prime`, который проверяет, является ли результат декорируемой функции (суммы чисел) простым числом, и печатает "Простое:" или "Составное:". | декораторы, проверка простоты чисел |
| `Practic.py` | Функция `gen_repeat`, возвращающая замыкание для повторения первых двух символов строки. Создание списка функций и их применение к одному и нескольким аргументам. Декоратор `memoize_func` для кэширования результатов. | замыкания, списки функций, мемоизация, `dict` как кэш |
| `large_file.txt` | Большой текстовый файл (отрывок из «Войны и мира» на русском) для тестирования построчного чтения. | тестовые данные для генераторов |

## 🚀 Как запустить любой скрипт

1. Откройте терминал в папке `ModulsOfTema1/Module9`.
2. Выполните:
   ```bash
   python Lection1.py
   python Lection2.py
   python Lection3.py
   python Lection4.py
   python Lection5.py
   python Lection6.py      # требует large_file.txt в той же папке
   python Lection7_decorators.py
   python Lection8_decorators.py
   python Module9_HW_1.py
   python Module9_HW_2.py
   python Practic.py
   ```

> **Примечание**: `Lection6.py` читает файл `large_file.txt` (предоставлен). Убедитесь, что он находится в той же директории.

## 📦 Зависимости

Все скрипты используют только стандартную библиотеку Python. Дополнительных установок не требуется.
