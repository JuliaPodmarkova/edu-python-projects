# Модуль 8: исключения (Exception Handling)

В этой папке собраны программы, демонстрирующие механизм обработки исключений в Python: возбуждение (`raise`), перехват (`try-except`), блоки `else`, `finally`, создание пользовательских классов исключений, а также практическое применение для валидации данных и устойчивости программ к некорректному вводу.

## 📂 Содержание

| Файл | Что делает | Демонстрирует |
|------|------------|----------------|
| `test.py` | Базовые примеры перехвата исключений: `ZeroDivisionError`, `NameError`, `OSError` при открытии файла. Работа блоков `else` и `finally`. Цепочки вызовов функций с исключениями. | `try-except`, `else`, `finally`, вложенные `try`, передача исключений по стеку вызовов |
| `test1.py` | Создание пользовательского исключения `ProZero` (наследник `Exception`). Генерация исключения через `raise` с дополнительными параметрами. Функция `greet_person` с проверкой имени. | пользовательские исключения, `raise`, атрибуты исключений (`message`, `extra_info`) |
| `HomeWorkM8.py` | Функция `personal_sum` обрабатывает `TypeError` внутри итерации, считает сумму чисел и количество некорректных элементов. `calculate_average` использует её, обрабатывая деление на ноль и тип аргумента. | обработка ошибок при работе с коллекциями, `try-except` внутри цикла, возврат кортежа |
| `HomeWorkM8.2.py` | Классы `IncorrectVinNumber` и `IncorrectCarNumbers` – пользовательские исключения. Класс `Car` в конструкторе проверяет VIN (тип `int`, диапазон 1000000–9999999) и номер (тип `str`, длина 6). При ошибке возбуждается соответствующее исключение. | пользовательские исключения, валидация данных, `isinstance`, `raise`, перехват разных типов |
| `practic.py` | Чтение файла `data.txt` со строчками вида `число операция число`. Функция `calc` разбирает строку, выполняет арифметическую операцию и печатает результат. Обрабатываются `ValueError` при распаковке или преобразовании в `int`. Используется форматирование `format()` для номера строки. | обработка ошибок при парсинге, работа с файлом, перехват `ValueError`, анализ сообщения об ошибке (`exc.args`) |
| `data.txt` | Тестовый файл с примерами выражений (в том числе некорректными: `- 5 6`, `W + 15`). | входные данные для `practic.py` |

## 🚀 Как запустить любой скрипт

1. Откройте терминал в папке `ModulsOfTema1/Module8`.
2. Выполните:
   ```bash
   python test.py
   python test1.py
   python HomeWorkM8.py
   python HomeWorkM8.2.py
   python practic.py          # требует наличия data.txt в той же папке
   ```

> **Примечание**: `practic.py` читает файл `data.txt` (предоставлен) – убедитесь, что он находится в той же директории.

## 📦 Зависимости

Все скрипты используют только стандартную библиотеку Python, дополнительных установок не требуется.
